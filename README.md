## ğŸ¨ Customization

### Colors
The website uses a beautiful gradient color scheme. You can customize colors in `styles.css`:
- Primary gradient: `#667eea` to `#764ba2`
- Accent colors: `#ff6b6b`, `#4ecdc4`, `#45b7d1`, `#f9ca24`, `#ff9ff3`

### Animations
All animations are defined in CSS with customizable durations and effects:
- Balloon floating: `balloon-float` animation
- Confetti falling: `confetti-fall` animation
- Text bouncing: `bounce` animation
- Cake glowing: `cake-glow` animation

### Interactive Elements
Modify the JavaScript in `script.js` to:
- Change confetti colors and patterns
- Adjust animation timings
- Customize game difficulty and scoring
- Add new interactive features
- Customize celebration effects

### Game Features
The balloon pop game includes:
- 10-second timer with real-time countdown
- Score tracking (10 points per balloon)
- Random balloon spawning with different colors
- Pop animations with particle effects
- Encouraging messages based on performance
- **Special Birthday Message Page**: Automatically redirects to personalized message after game ends
- Restart and close game functionality

### Birthday Message Page
After the game ends, players are taken to a beautiful dedicated page featuring:
- Personalized "Happy Birthday Dhwani" message
- Photo display (add `dhwani-photo.jpg` to project folder)
- Complete heartfelt birthday message with emojis
- Continuous confetti and floating hearts animations
- Game score display from previous game
- Navigation back to main game
